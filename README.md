# Fake Job Detection System

## Project Description
A Machine Learning based system that detects whether a job posting is genuine or fake using Natural Language Processing techniques.

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLP (TF-IDF)
- Logistic Regression
- Streamlit

## Features
- Accepts job description from users
- Predicts fake or genuine job postings
- Simple web interface

## How to Run

Install requirements:

pip install -r requirements.txt

Run the application:

streamlit run app.py
